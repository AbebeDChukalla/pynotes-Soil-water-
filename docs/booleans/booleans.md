
# Booleans


```python
# Test for equality
'DNA' == 'RNA'
```




    False




```python
# Test for inequality
'DNA' != 'RNA'
```




    True




```python
# AND conditional
var = 'Kansas'
var == 'Oklahoma' and var == 'Kansas'
```




    False




```python
# OR conditional
var == 'Oklahoma' or var == 'Kansas'
```




    True




```python
# Combined AND-OR conditional
# AND executes first
var == 'Kansas' and var == 'Oklahoma' or len(var) == 6
```




    True
